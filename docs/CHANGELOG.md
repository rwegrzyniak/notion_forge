## [0.1.0] - 2025-10-02

### Added
- Initial release of NotionForge
- Infrastructure as Code DSL for Notion
- CLI tool with Thor
- Modern Ruby 3.3+ support with pattern matching
- Parallel processing with Ractors (experimental)
- Fiber-based async operations
- Rich DSL for databases, pages, and templates
- Query builder with pattern matching
- State management for idempotent operations
- RuboCop configuration
- GitHub Actions CI/CD
- Comprehensive documentation

### Features
- Database creation with properties (title, text, status, select, etc.)
- Page creation with rich content blocks
- Template system for reusable structures
- Workspace management with dependency resolution
- Resource collections with lazy evaluation
- Refinements for enhanced Ruby syntax
- CLI commands: init, forge, validate, examples, clean
